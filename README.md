# 📚 Library Management System (C++)

---

## 1. 📝 Introduction
The Library Management System is a console-based application developed in **C++** using Object-Oriented Programming (OOP) concepts.  
This project helps manage library operations such as adding books, searching books, issuing, and returning books efficiently.

It reduces manual work and provides a simple system to maintain library records.

---

## 2. 🎯 Objectives of the Project
- Implement **Object-Oriented Programming** concepts  
- Manage **book records** efficiently  
- Perform operations like **add, delete, search, issue, and return books**  
- Understand the use of **vectors (dynamic arrays)**  
- Simulate a **real-world library system**  

---

## 3. 🛠 Methodology / System Design
The system is designed using **OOP principles**:

- **Class & Object** → `Book` and `Library` classes  
- **Encapsulation** → Data accessed using member functions  
- **Inheritance** → `IssuedBook` class inherits `Book` class  
- **Polymorphism** → `display()` function is overridden  
- **Data Structure Used** → **Vector** (dynamic storage)  

### 🔄 System Flow:
1. User selects an option from the menu  
2. The system performs the selected operation  
3. Data is stored and updated in vectors  
4. Menu repeats until the user exits  

---

## 4. 💻 Implementation / Source Code
The project is implemented in **C++** and includes the following features:

- **Add Book**  
- **View Books**  
- **Search Book**  
- **Delete Book**  
- **Issue Book**  
- **Return Book**  
- **View Issued Books**  

### 🛠 Technologies Used:
- **C++ Programming Language**  
- **Standard Template Library (STL - vector)**  

---

## 5. ✅ Conclusion
This project demonstrates how **Object-Oriented Programming** can be used to build a real-world application.  
It improves understanding of concepts like **inheritance, polymorphism, and data handling**.
